# 🚀 新建监控 - 快速入门

> 监控对象新建问题已完全修复！现在新建超级简单。

## ⚡ 最快的方式 (30秒完成)

```
1. 点击"新建"按钮清空表单
2. 填写 3 个字段:
   • 名称: 例如 "品牌A舆情"
   • 关键词: 例如 "品牌A"  
   • 平台: 默认已选 (weibo,zhihu,baidu)
3. 点"保存设置"
✓ 完成！
```

## 📝 各字段说明

### 基本配置
| 字段 | 必填? | 默认值 | 说明 |
|------|------|-------|------|
| 名称 | ✓ | 无 | 系统内唯一标识 |
| 分组 | ✗ | 未分组 | 方便管理 |
| 标签 | ✗ | 无 | 自定义标签，逗号分隔 |

### 采集配置
| 字段 | 必填? | 默认值 | 说明 |
|------|------|-------|------|
| 平台 | ✓ | weibo,zhihu,baidu | 监控的社交平台 |
| 关键词 | ✓ | 无 | 采集的关键词，逗号/换行分隔 |
| 采集量 | ✗ | 60 | 每个平台每次采集多少条 |

### 执行配置
| 字段 | 必填? | 默认值 | 说明 |
|------|------|-------|------|
| 间隔(秒) | ✗ | 1800(30分钟) | 多久执行一次 |
| 启用 | ✗ | ✓ 启用 | 是否自动运行 |
| 报告格式 | ⚠️ | ✓ DOCX+PDF | 生成报告的格式 |

### 告警阈值
| 字段 | 默认值 | 说明 |
|------|-------|------|
| 负面占比 | 0.3(30%) | 负面内容超过此比例触发告警 |
| 风险分 | 50 | 综合风险评分阈值 |
| 采集量 | 30 | 采集条数少于此值触发告警 |

## ✨ 新建的智能特性

- ✅ **自动默认值**: 新建时所有字段都有合理的默认值
- ✅ **容错机制**: 即使忘记选择报告格式，也会用默认值创建
- ✅ **智能初始化**: 新建时自动勾选常用的格式(DOCX+PDF)
- ✅ **清晰提示**: 必填项缺失时有清楚的提示

## ❓ 常见问题

**Q: 报告格式不选会怎样?**
A: 会自动使用默认值(DOCX和PDF)，无需担心

**Q: 不填关键词可以吗?**
A: 不可以，必须至少填一个。会收到清楚的提示。

**Q: 创建后可以修改吗?**
A: 完全可以。点击列表中的监控，修改后保存即可。

**Q: 禁用的监控会怎样?**
A: 不会自动执行，但可以手动点"立即执行"按钮

**Q: 关键词怎么格式化?**
A: 用逗号分隔 (品牌A,品牌B) 或每行一个都可以
